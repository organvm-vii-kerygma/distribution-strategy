## [0.3.0] - 2026-02-24

### Added

- Data export module (`kerygma_strategy.data_export`) generating distribution plan and channel report
- `distrib-export` CLI entry point for artifact generation
- `data/distribution-plan.json` — channel inventory, calendar events, scheduling config
- `data/channel-report.json` — platform capabilities, analytics and report schema
